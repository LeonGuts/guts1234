# guts1234
Тестирование "Заметки" на Iphone SE 2, IOS 26.3.1

ТЗ: 
Выбери любое мобильное приложение на своём устройстве (например, мессенджер, заметки, календарь, банк и т.п.).
Определи ключевые пользовательские функции приложения (создание, редактирование, удаление, регистрация, вход и др.).
Составь 10 тест-кейсов для этих функций: укажи название, шаги и ожидаемый результат (оформи списком или таблицей в .xlsx либо Markdown).
Постарайся покрыть как позитивные, так и негативные сценарии (валидные и невалидные действия пользователя).
Результат отправь с помощью формы тестового задания. Спасибо!
Внимание: не используй в названии репозитория, коммите, папке или коде название тестового задания или сайта — это может привести к отмене заявки. Будь внимателен!
